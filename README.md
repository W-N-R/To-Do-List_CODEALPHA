# To-Do-List_CODEALPHA


Description:
This is a simple console-based application written in C++ that helps users manage their daily tasks. It allows you to add tasks, view the list, mark tasks as completed, and remove them when they are no longer needed.

Features:
1. Add Task: Input a description to create a new task.
2. View Tasks: See all current tasks with their status (completed or pending).
3. Mark Task as Done: Updates the status of a specific task to [x].
4. Remove Task: Deletes a task from the list permanently.

How to Run:
1. Make sure you have a C++ compiler installed (like g++).
2. Save the code file as 'todo.cpp'.
3. Open your terminal or command prompt.
4. Compile the code using the command: g++ todo.cpp -o todo
5. Run the application using: ./todo (on Linux/Mac) or todo.exe (on Windows).

Requirements:
- C++ Compiler (GCC, Clang, or MSVC)
- Standard C++ libraries (iostream, vector, string)
